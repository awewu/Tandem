## 九、关键认知 / 不可丢

1. **Tandem ≠ SaaS**. 是 Owner 自己公司的内部协作 AI 平台. 飞书路径 (内部 4 年再考虑对外).
   - 不做: 定价 / GTM / SSO / 国际化 / App Store
   - 做: 同事自用 → 70%+ 周打开 ≥ 3 次 → 再谈商业化

2. **Tandem AI = 中央 AI = 老板分身**. 永远在线, 方向不明就问.
   - 永远不替员工签字 (重大决议必走议事室)
   - 给方向 + 优先级 + 判断框架

3. **OKR 是第一性原理**. 不是"组织 IQ 放大器"那么抽象, 是为 OKR 而活.
   - 任何议题 / 任务 / 决议必须可回溯到当前 OKR
   - CompanyBrain system prompt 永远先注入 OKR Anchor

4. **UI 宪章不可妥协** (`docs/CHARTER-UI-V1.md`).
   - 颜色: 走 CSS var, 不 raw Tailwind (slate/amber/rose)
   - 字体: text-{display/title-1-3/headline/body/caption/footnote}
   - 阴影: shadow-soft-* (不 shadow-sm/md/lg)
   - Hero: hero-ink / glass, ≥ text-title-2

5. **2026 Anthropic best practice 落地**: 自写 agent loop + Subagent + Compaction + Production Evals.

6. **2026 Mem0 best practice 落地**: 4-scope memory + Procedural kind + Async writes + Reranker.
