# Tandem 同事入门指南 · v0.1

> 这份指南是给公司同事第一次打开 Tandem 时看的。
> 5 分钟读完，就能开始用。

---

## 一、Tandem 是什么

**Tandem = 我们公司内部用的 AI 协作平台**，把 OKR、议事、1on1、个人成长 拧成一根绳。

不是 SaaS 软件，**只我们公司自己人用**。所有数据存我们自己服务器，不出公司。

---

## 二、3 个必学，30 分钟搞定

### 1. 登录 · 1 分钟

- 浏览器打开 `https://tandem.公司域名/`（也可以装到桌面：Chrome 右上角"安装 Tandem"）
- 用公司邮箱登录
- 手机也能用 — 装 PWA 当原生 App

### 2. 「问老板」· Tandem AI 永远在线 · 2 分钟

**任何页面右下角**有一个 ✨**问老板** 浮动按钮（或按 ⌘J / Ctrl+J）。

点开，你能：

- 方向不明 → 「我这周该聚焦什么 OKR?」
- 决策卡壳 → 「这个客户该不该接?」
- 战略疑问 → 「这件事怎么对齐公司战略?」
- 1on1 准备 → 「我这周该和谁聊?」

回答基于：**老板的决策思路 + 当前公司 OKR + 公司 SOP**。

> ⚠️ 它是老板的**分身**，不是老板本人。给你方向 / 优先级 / 判断框架，**不替你签字**。重大决议还是要走议事室。

所有问答**自动入审计**（公司治理需要），但只有你和治理委员会能看，同事看不到。

### 3. 我的 OKR · 3 分钟

- 进 `/okr` 看自己的目标
- 每条 KR 显示进度 + 风险（绿/黄/红）
- **每天花 5 分钟写日报** (`/report`) — AI 帮你聚焦今天推进了哪几个 KR

> 公司铁律：**任何议题 / 任务 / 决议都要回溯到至少 1 个 OKR**。
> 不知道怎么对齐？「问老板」。

## 四、5 个关键模块

### `/atlas` · 平台地图

不知道某个功能在哪里，先看 atlas。

### `/okr` · OKR 中心

公司 / 部门 / 个人 三层 OKR 联动。点任一 KR 看进展 + 历史 check-in。

### `/convergence` · 议事室

**Tandem 的核心**。任何方向不明的事，建议起议题，3+1 选项让 AI 帮你"逼出"决策。
17 分钟内收敛，所有决议自动锚 OKR + 进审计。

### `/persona` · 我的学员档案

你在 Tandem 里有一个**主分身（学员证）**。
- 5 个主修方向 GPA (设计 / 产品 / 技术 / 营销 / 战略)
- 学习进度 / 实习权限 (L0-L3)
- **训练你的分身越久越拿手** — 它能代你做的事越多

### `/learning` · 学院（v0.2 上线）

公司必修课 + 选修课 + 你自己自学的"学分置换"。
> 当前 v0.1 是骨架，真正课程内容会在 v0.2 开放。

## 五、隐私 · 你必须知道的 3 件事

> 公司宪章 §13：**数据归公司，尊严归员工**。

1. **同事看不到你的 1on1 / 个人 Memory / 私聊**
   - 默认私有。要分享必须你主动 share。

2. **AI 代你做事必须你 24h 内能撤回**
   - 任何 AI "代签" 行为先 draft，给你 24h 否决窗。

3. **你能导出 / 删除 / 匿名化自己的数据**
   - `/persona/data-source` 看你的数据流。
   - `/settings/privacy` 一键导出 / 注销。

## 六、问题怎么办

| 问题 | 怎么办 |
|---|---|
| 不知道怎么用 | 浮动「问老板」⌘J 直接问 |
| 觉得 AI 答错了 | 答案下方 👎 反馈, 进治理委员会月审 |
| 数据敏感想隐藏 | `/settings/privacy` 看 + 改 |
| 提个新功能 | 在 `/intranet/forum` 发帖 |
| 想培训分身 | `/persona/training` 给它喂决议历史 |
| 遇到 Bug | IM 群 #tandem-bug |

---

## 七、不要做的事 ❌

- ❌ 把 Tandem 的截图发到外部群（含公司战略 / 决议 / Persona）
- ❌ 把别人的 Persona 训练数据导出（数据归员工自己）
- ❌ 用「问老板」代替跟真人 1on1（AI 是搭子，不是上级）
- ❌ 把 AI 的回答当签字（重大决议必须走议事室）

## 八、术语速查

| 词 | 意思 |
|---|---|
| **事半** | OKR 驱动模块（牛马, 严格版） |

| **OKR Anchor** | 任何议题必须锚定的 OKR |

| **Skill Gateway** | 外部 AI 进 Tandem 的 4 道闸 |
| **17 分钟法则** | 议事室收敛上限 |
| **24h 否决窗** | AI 代签后你能撤回的时间 |
| **Steward** | 知识治理官（不是老板，独立角色） |

## 九、首周 KPI（你的）

不强制，但建议第一周达成：

- [ ] 完成 onboarding 学院（10 分钟）
- [ ] 设了 1 个 Q2 个人 KR
- [ ] 写了 3 篇日报
- [ ] 「问老板」用过 5 次
- [ ] 完成 1 次 1on1（跟上级 or 跟下属）
- [ ] 在议事室收敛 1 个议题
