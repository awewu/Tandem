# 牛马搭子 (Tandem) 用户使用指南

> V1 版本 · 给最终用户的快速上手手册

## 5 分钟快速上手

### 1. 第一次登录
- 通过钉钉/企微/飞书扫码登录 (无需注册)
- 系统自动为你创建 "拿捏老板分身" (新生 🥚 阶段)

### 2. 发起第一个议事
点击侧边栏 **议事室** → **发起议事**

```
议题示例:
  ✅ 客户 X 续约谈判策略
  ✅ Q2 团队是否扩招 2 人
  ❌ 周会议程 (这种不需要议事室)
  ❌ 闲聊 (不属于决议)
```

### 3. 选择 3+1 方案
系统自动生成 4 个选项:
- **A · SOP**: 按公司既有标准操作执行
- **B · AI 推演**: AI 综合分析的方案
- **C · 历史案例**: 类似历史决议的做法
- **D · 你的原创**: ⚠️ 必须人工填写, AI 不会代写

### 4. 提交决议
17 分钟内完成. 提交后:
- 24 小时内你可以"反悔"撤回
- 24 小时后自动生效, Action items 进入跟踪

## 核心概念

### 拿捏老板分身 (5 阶段)

```
🥚 新生 (0-2w)        仅旁听学习
🐣 学徒 (2w-2m)       代汇报数据
🐤 助手 (2m-6m)       绿区会议表态
🦅 副手 (6m-1y)       承诺 1 工作日动作
🐉 搭档 (>1y)         跨企业代行
```

### 议事室 5 步流程

```
1. 上下文收集    AI 自动找 SOP / 案例 / 关联 KR
2. 3+1 选项      A: SOP   B: 推演   C: 历史   D: 你的原创
3. 审议          团队讨论
4. 收敛          选定 + Action items
5. 提交          24h 内可否决, 之后生效
```

### OKR vs TTI (双轨)

```
KPI (硬指标)         TTI (成长度)
↓                    ↓
完成度 100%          完成度 60-70% 健康
挂钩奖金/调薪/末位   永不挂钩任何金钱回报 (含系数浮动, 宪章 §4)
按部门 / 岗位         按个人成长方向
```

## 我的数据归谁?

**重要**: 你的所有数据归 **公司所有**, 但有 4 项保障:

1. **导出权**: 你可随时导出个人原始数据 (会议录音/邮件)
2. **匿名化**: 离职后, 公司必须匿名化你的画像
3. **否决权**: AI 替你做的任何决策, 24h 内你可撤回
4. **拒绝代笔**: 高敏内容 (薪资/法律) AI 不会代写

## 常见问题

### Q: 议事室必须 17 分钟内结束吗?
A: 是. 17 分钟硬上限, 超时自动转给主管做最终决定. 这是为了防止"会而不议".

### Q: D 选项一定要填吗?
A: 不一定. 但 D 是给你"独立思考空间"的. 如果你只信 SOP / AI / 历史, 长期会丧失判断力.

### Q: 拿捏老板分身会不会假装我?
A: 不会. 任何 AI 代行都强制水印 "🤖 AI 代行". 高敏话题 (薪资/裁员/法律) 自动拒绝代笔.

### Q: 我可以拒绝创建分身吗?
A: 可以. 在 `/persona` 页面关闭 "学习活跃" 即可暂停学习. 已学习的数据可申请删除.

### Q: 如何申请成为 Steward (知识管家)?
A: 联系部门负责人提名. Steward 是有"利益冲突"识别能力的资深员工, 把守 Memory 入库的最后一关.

## 进阶: Memory 升级流程

如果你发现一个**优秀做法**应该入公司知识库 (Memory):

1. 在 `/memories` 找到对应 Material
2. 点 "提议升级为 Memory"
3. 选类型: SOP / Case / Redline / Value
4. 等业务负责人 + Steward + CEO 三方签字
5. 公示 7 天 (员工可异议)
6. 自动生效 → 全公司可引用

紧急通道: CEO 签字 + 24h 公示期减半
